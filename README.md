# Softgames worktest


## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build
```

## Running
```sh
npm install http-server, run it inside dist and open the ip in your browser.
```

```sh
node dist/bundle.js
```
